# Himalaya Trails Admin

Himalaya Trails Admin is a React learning project that simulates a modern travel agency management dashboard for Nepal. The application is designed to help administrators manage travel packages, bookings, customers, destinations, and business analytics through a clean and responsive interface.

The project is built to practice modern React development concepts while creating a real-world dashboard application.

---

## About the Project

This project focuses on building an admin dashboard for a Nepal-based travel agency.

The application will allow administrators to:

- Manage travel packages
- View and organize bookings
- Manage customer information
- Maintain destinations
- View business analytics
- Search and filter records
- Practice CRUD operations using dummy data
- Learn global state management with Context API

Initially, all data will be stored locally using JavaScript objects. In future versions, the dashboard can be connected to a Laravel backend and MySQL database.

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/himalaya-trails-admin.git
```

### Navigate into the project

```bash
cd himalaya-trails-admin
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
